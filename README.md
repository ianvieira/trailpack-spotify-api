Trailpack to connect your app with the Spotify API

## Install

```sh
$ npm install --save trailpack-spotify-api
```

## Configure

```js
// config/main.js
module.exports = {
  packs: [
    // ... other trailpacks
    require('trailpack-spotify-api')
  ]
}
```
